Panel de Control de Aulas

Una aplicación web interactiva Single-Page (SPA) diseñada para la gestión centralizada de recursos en instalaciones educativas.

Características Clave:

  • Sistema de Autenticación: Flujo de registro e inicio de sesión para alumnos validado a través de matrícula, simulando una base de datos con localStorage y manejo de sesión activa con sessionStorage.
  • Generación Dinámica de Interfaz: Creación automatizada mediante JavaScript de tarjetas de control para 25 áreas distintas (21 Salones, SUM, Salas de Cómputo y Sala de Maestros).
  • Panel de Hardware Simulado: Botones de acción integrados para el control de Clima (encendido, apagado, cambio de temperatura) y gestión de TV (encendido/apagado, solicitud de asistencia HDMI).
  • Buzón de Sugerencias Inteligente: Formulario interactivo que incluye un filtro de moderación de lenguaje (array de palabras no permitidas) y un algoritmo que limita el uso a 3 sugerencias diarias por alumno según la fecha del sistema.
